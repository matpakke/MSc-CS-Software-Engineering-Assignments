MITRE ATT&CK Framework

Core ATT&CK Tactics

Tactic Description
Initial Access Techniques that adversaries use to gain an initial foothold within a network.
Execution Techniques that result in adversary-controlled code running on a local or remote system.
Persistence Techniques that adversaries use to maintain their foothold.
Privilege Escalation Techniques that adversaries use to gain higher-level permissions on a system or network.
Defense Evasion Techniques that adversaries use to avoid detection throughout their compromise.
Credential Access Techniques that adversaries use to steal credentials, such as account names and passwords.
Discovery Techniques that allow adversaries to gain knowledge about the system and internal network.
Lateral Movement Techniques that adversaries use to move through the network.

Example Techniques

Technique Tactic Description
Phishing Initial Access Tricking users into providing credentials or running malware.
PowerShell Execution Using PowerShell commands to execute scripts and code.
Create Account Persistence Creating new accounts to ensure continued access.
Process Injection Defense Evasion Injecting code into processes to hide execution.
Credential Dumping Credential Access Extracting credentials from a system to use later.
Remote Desktop Protocol (RDP) Lateral Movement Using RDP to move to other systems on the network.
MITRE ATT&CK helps in identifying and defending against cyber threats.
